# AI Applications Portfolio: Autonomous Systems with Local LLMs

Based on the blog post analysis from the .har file, this is a comprehensive portfolio of autonomous AI applications built by Daniel Kliewer, focusing on local LLM inference, MCP protocols, and GraphRAG architectures.

## Portfolio Overview

**Core Technologies**: Local LLMs (Ollama), MCP Protocol, GraphRAG, Reinforcement Learning, Distributed Agent Systems

**Key Applications**:
- **SpecGen**: Deterministic code generation via agentic RAG
- **MCBot01**: Local-first full-stack foundation for LLM development
- **PersonaGen**: Autonomous persona generation system
- **Orthos**: Self-correcting agent validation framework

**Principles**:
- Computational sovereignty through local inference
- Deterministic architectures over probabilistic generation
- Memory preservation via GraphRAG
- Distributed agent coordination via MCP protocols
- Self-improvement through reinforcement learning loops

## Technical Architecture

**Inference Layer**: Ollama for local LLM execution
**Communication**: MCP (Model Context Protocol) for agent coordination
**Memory**: GraphRAG for persistent, structured context
**Validation**: AST parsing, import resolution, and self-correction mechanisms
**Deployment**: Containerized local execution with optional cloud integration
